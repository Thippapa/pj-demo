# Lead Weight Dashboard

A web-based dashboard for recording and monitoring lead weight data in a manufacturing environment.

## Project Overview

This project is designed to support manufacturing data collection and monitoring.  
Users can record lead weight data by date, department, production line, lead type, and weight.

The dashboard provides KPI summary, filtering, charts, and drill-down analysis.

## Key Features

- Data entry form with validation
- Dashboard with KPI cards
- Filter by year, month, department, and line
- Charts: daily, monthly, department, lead type
- Drill-down interaction
- Export to Excel and PDF

## Tech Stack

- HTML
- CSS
- JavaScript
- Chart.js
- localStorage

## Data Flow

User Input → Validation → localStorage → Data Processing → Dashboard
